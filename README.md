# Final-Project
The parser converts MEI files into a normalized internal model that can be used by the web application and database layer.

Technology used:

BACKEND
Node.js
Express
PostgreSQL (pg)
dotenv
cors
xml2js

FRONTEND
React
Vite

TESTING
Jest

MEI Samples Files taken from the verified Github Account of the Music Encoding Initiative
https://github.com/music-encoding
https://github.com/music-encoding/sample-encodings/tree/main/MEI_5.1
